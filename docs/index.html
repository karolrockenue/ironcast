IRON CAST
STRENGTH TRAINING LOG · iOS

WHAT IT IS

IronCast is a personal strength-training logger. It runs a strict two-workout split (A and B), alternated automatically based on the last completed session. Auto-suggests weights on each top set, runs a per-exercise rest timer in the background, and detects PRs without any input from you.

FEATURES

  • Strict A/B alternation — no calendar, no configuration.
  • Two working sets per exercise. Set 1 top set, Set 2 back-off.
  • Automatic progression on top-set reps.
  • Deadlift heavy/technique alternation every B session.
  • Per-exercise rest timer with background notification.
  • Mid-workout PR detection (weight, rep, volume).
  • End-of-session summary with PRs and progression recap.

PRIVACY

IronCast collects no data. There is no account, no cloud sync, no analytics, and no ads. Your workout history lives in a single SQLite file on your device. Uninstall the app and it is gone.
