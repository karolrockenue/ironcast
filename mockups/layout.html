States visible here

  • Done — green tint, checkmark, set results inline as chips. Card is collapsed, not interactive.
  • Active — accent border + faint blue gradient. Sets expanded. The first unlogged set is highlighted as current. Drag its slider to log.
  • Pending — dimmed. Tap to jump there (if you want to reorder on the fly). Otherwise the app auto-advances when the active exercise is fully logged.

Try it

  • Drag the Lat Pulldown Set 2 slider — the set row flips to “logged”. Log both sets in-range and the card auto-transitions to done, and Seated Cable Row becomes active.
  • Tap any pending card to manually activate it instead.
  • The Mark done button in the active card lets you skip remaining sets (e.g. if the rack is taken) without losing the ones you did.
